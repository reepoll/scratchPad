Exercise 3

Attach a listener to the button below. When the button is clicked, take the value from the input field with id ex3-text-to-hash and get the md5 hash of it by making an Ajax call to http://md5.jsontest.com/?text=example_text. When you make the ajax call, you'll replace example_text with the text from the input field. Take the result from the JSON response and put it into the field with id ex3-hash-result.

Watch example behavior: http://i.imgur.com/y9eDv5U.gif

Relevant documentation:

  • jQuery click
  • encodeURIComponent
  • jQuery ajax
  • jQuery val
